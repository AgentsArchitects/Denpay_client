## Getting Started

### Installation

```bash
cd client-ui
npm install
```

### Development

```bash
npm run dev
```

The application will run on `http://localhost:5174`

### Build

```bash
npm run build
```

## Design System

The UI follows a consistent design language with:

- **Primary Color**: Purple (#8B5CF6)
- **Accent Colors**: Blue (#3B82F6), Green (#22C55E)
- **Typography**: Inter (body), Poppins (headings)
- **Animations**: Smooth transitions and floating elements
- **Shadows**: Subtle elevations for depth
